# Students-habits-and-performance
Objectives
Understand how daily lifestyle choices affect academic success
Identify habits that positively or negatively correlate with performance
Visualize trends and patterns using Python
Provide insights for students, parents, and educators

Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

 1. Data Loading & Initial Exploration
Importing dataset
Checking structure, missing values, and data types
Early observations

 2. Data Cleaning
Handling missing values
Converting data types (e.g., sleep hours, study hours)
Removing outliers where necessary

 3. Exploratory Data Analysis (EDA)

Key relationships explored:
Sleep hours vs Grades
Study time vs Performance
Social life vs Academics
Exercise habits
Screen time patterns

 4. Visualizations
The notebook includes:
Correlation heatmap
Scatter plots
Distribution plots
Trend analyses
Category comparisons
